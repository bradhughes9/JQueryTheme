What is Responsive Web Design?

In Responsive Web Design the goal is to make your website flexible. When I say flexible what I mean is that you want to be able to view your website on a desktop the same way that you can view it on a mobile device such as a cell phone or tablet. In order to apply responsive web design to your websites you just need to apply CSS to certain parts of your page and include some media queries. Doing this will allow your content on your website to shrink or expand with the varying sizes in viewports from devices.

Principles

Grid View is one of the most important principles in webdesign, and it's a fairly simple method. Grid View is when you design your website to be set in a grid like design using columns. To do this you will Set a class on each div that you will want to make a different coulmn and assaign it the percentage of the width of the webpage for desired size. You can do this as many times as you want but the more columns you have the smaller each column will be. Each column does not have to be the same width for example you could have one column set to 80% and another set to 20%, or have two columns each set to 50%. Having your iamges flexible as well is another important part of making a website Responsive. When your page shrinks down to a mobile device's viewport and all the content on your page shrinks except for the images on your page, it will look very odd. You can make images flexable the same way you make your page flexible by adjusting the width to a percentage and the height to auto. This will allow your iamge to scale witht eh height and width responsively. There is another way to make your images responsive by using the max-width setting, which will shrink the image but never make the image larger than it's original size.

Media Queries are very essential for web designers when building Responsive Web pages, because they can have the website automatically switch to a different layout that they design all depending on the viewers screen size. This makes is very easy for a web designer to choose what the content will look like each different screen size. They can choose to the orientation of the page, screen resolution, color, and even hide iamges.
